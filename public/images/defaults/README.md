# Section Default Images

Drop JPG/PNG files here named after section slugs to give every post in that section a default card image when the post itself doesn't set `cover_image` in front matter.

Filenames (16:9 ratio, ~1600x900 recommended):

- posts.jpg              (main Daily Brief)
- uncle-ramesh.jpg
- rumor-mill.jpg
- speedeet-wilar.jpg
- dj-roadblock.jpg
- bam-bam-sally.jpg
- back-a-truck.jpg
- de-boys-seh.jpg
- bounty-board.jpg
- progress-report.jpg
- patriots-portfolio.jpg

To override on a per-post basis, add to post front matter:

    cover_image: "/images/posts/my-story.jpg"

Place the image in static/images/posts/ and Hugo will serve it.

SAFETY: Only use public-domain, CC-licensed, or images you own.
Photos of named individuals must pass the Two-Gate Test in the
Tradewinds Image Safety Protocol before use.

Good free sources:
- Wikimedia Commons (CC/public domain)
- Unsplash (free commercial use)
- Pexels (free commercial use)
- Pixabay (free commercial use)
- Government press releases (public domain in most jurisdictions)
